# ⚽ FIFA World Cup 5-Tournament Historical Analytics (2002–2018)
**Power BI Capstone Project** | *Author:* Janefrances Chinaza Oodo

---

## 📌 Executive Summary
This repository contains an end-to-end Business Intelligence capstone project analyzing 5 consecutive FIFA World Cup tournaments (2002–2018). Using **Microsoft Power BI**, multi-year tournament data was modeled into a Star Schema architecture to evaluate scoring dynamics, match outcome distributions, and national team performances—with special analytical spotlights on **France** and **Nigeria**.

---

## 🛠️ Technical Stack & Methods
* **Business Intelligence:** Microsoft Power BI Desktop
* **Data Modeling:** Star Schema Architecture, Dimensional Modeling
* **Calculations:** Data Analysis Expressions (DAX) Measure Library
* **Documentation:** Executive Case Study Report & Stakeholder Presentation Deck

## 🔑 Key Analytical Insights
* **Goal Trajectories:** The 2014 World Cup in Brazil recorded the highest scoring output across the 5-tournament window (171 goals).
* **Dominant Performers:** Germany (69 pts, 64 goals) and Brazil (63 pts, 56 goals) led overall standings.
* **Target Spotlight (Nigeria):** Qualified for 4 out of 5 tournaments, securing 13 total matches (2 Wins, 3 Draws, 8 Losses, 9 Points) with analysis highlighting defensive performance trends during group stages.
